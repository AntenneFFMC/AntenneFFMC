La structure de votre site est construite sur une hiérarchie de rubriques. Une rubrique peut contenir autant de sous-rubriques que nécessaires. Il est important que le nombre de rubriques principales soit limité car elles sont trop serrées au-delà de 4.

La fédération recommande une homogénéité des titres et ordre de rubriques. Voici les rubriques de base qui doivent être présentes sur tous les sites antennes :
1. "NOS ACTIONS",
2. "LA FFMC SAVOIE",
3. "LE MOUVEMENT"

Les rubriques à afficher dans le menu principal se configurent dans la navigation

Les autres rubriques sont référencées uniquement dans le plan du site.

Il est possible d’afficher la rubrique de votre choix sous la forme d’un widget vignette.

Les rubriques qui ne contiennent pas d'article n'apparaissent naturellement pas publiquement.
